# Mioplan Project

## Структура проекта

```
/
├── client/              # Фронтенд часть приложения
│   ├── src/             # Исходный код фронтенда
│   ├── package.json     # Зависимости фронтенда
│   └── vite.config.js   # Конфигурация Vite
├── server/              # Серверная часть приложения
│   ├── config/          # Конфигурационные файлы
│   ├── models/          # Модели данных
│   ├── routes/          # Маршруты API
│   ├── server.js        # Точка входа сервера
│   └── package.json     # Зависимости сервера
├── vercel.json          # Конфигурация для деплоя фронтенда на Vercel
├── DEPLOYMENT.md        # Инструкции по развертыванию
└── README.md            # Документация проекта
```

## Описание частей проекта

### Фронтенд (client/)
Фронтенд часть приложения реализована на React с использованием Vite. Она предоставляет интерфейс для работы с досками и задачами в виде матрицы важности и сложности.

Основные функции:
- Визуализация задач в виде матрицы 3x3 (важность/сложность)
- Возможность перетаскивания задач между ячейками
- Просмотр задач во временной шкале
- Интеграция с собственным API

URL фронтенда: https://mioplan.vercel.app/

### Сервер (server/)
Серверная часть приложения реализована на Node.js с использованием Express.js. Она предоставляет API для работы с досками и задачами.

Основные функции:
- Хранение данных о досках и задачах в PostgreSQL
- Предоставление REST API для фронтенда
- Проксирование запросов к Kaiten API

URL сервера: https://mioplan.onrender.com

## Команды

### Для запуска сервера:
```
cd server
npm start
```

### Для запуска сервера в режиме разработки:
```
cd server
npm run dev
```

### Для запуска фронтенда в режиме разработки:
```
cd client
npm run dev
```

### Для сборки фронтенда:
```
cd client
npm run build
```

## После деплоя

После деплоя сервера на Render:
1. Обновите конфигурацию прокси в `client/vite.config.js`, чтобы она указывала на URL вашего сервера
2. Обновите CORS настройки в `server/server.js`, добавив туда URL вашего фронтенда на Vercel
3. Передеплойте обе части приложения
