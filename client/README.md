# Kaiten Matrix Frontend

React-приложение для отображения задач из Kaiten API на интерактивной матрице важности и сложности.

## Описание проекта

Приложение позволяет визуализировать задачи из Kaiten в виде матрицы 3x3, где:
- Горизонтальные оси матрицы — Сложность: Сложно, Средне, Просто (слева направо).
- Вертикальные оси — Важность: Важно, Средне, Минимум (сверху вниз).

Каждая задача отображается как отдельная карточка с названием, описанием и метками в соответствующей ячейке матрицы.

## Технологии

- React 18
- Vite
- Axios
- CSS

## Установка и запуск

### Локальная разработка

1. Установите зависимости:
```bash
npm install
```

2. Запустите приложение:
```bash
npm run dev
```

### Настройка API

При локальной разработке приложение ожидает, что сервер запущен на `http://localhost:3001`.

После деплоя на Vercel, вам нужно будет обновить прокси в `vite.config.js`, чтобы он указывал на ваш сервер на Render.

## Сборка

Для сборки проекта выполните:
```bash
npm run build
```

Собранные файлы будут находиться в папке `dist`.
